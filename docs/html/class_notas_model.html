NotasModel::actualizarNotas ( $id_alumno,
$id_curso )

Actualiza las notas de módulos y cursos según las unidades formativas registradas.

Calcula y actualiza la calificación promedio de las notas de tipo 'Unidad_Formativa' para actualizar la nota de cada módulo y, posteriormente, la nota promedio de los módulos para actualizar la nota final del curso.

Parameters
string$id_alumnoIdentificador del alumno.
string$id_cursoIdentificador del curso.
Returns
void

◆ getModulosUnidadesPorCurso()

NotasModel::getModulosUnidadesPorCurso ( $idCurso)

Obtiene los módulos y unidades formativas asociados a un curso.

Realiza una consulta que une dos conjuntos de resultados mediante UNION ALL:

  • Los módulos asociados al curso.
  • Las unidades formativas asociadas a cada módulo del curso.
Parameters
string$idCursoIdentificador del curso.
Returns
array Array de registros que incluyen ID, Nombre, Tipo ('Modulo' o 'Unidad_Formativa') y, en el caso de unidades formativas, el ID del módulo asociado.

Inserta una nota en la base de datos.

Registra una nueva nota para un alumno en un curso, especificando a qué módulo o unidad formativa pertenece la nota, el tipo de nota y la calificación obtenida. La fecha de registro se establece como la fecha actual.

Parameters
string$id_alumnoIdentificador del alumno.
string$id_cursoIdentificador del curso.
string$id_moduloIdentificador del módulo.
string$id_unidad_formativaIdentificador de la unidad formativa.
string$tipo_notaTipo de nota ('Unidad_Formativa', 'Modulo', 'Curso', etc.).
float$calificacionCalificación obtenida.
Returns
bool True en caso de éxito, false en caso de error.
