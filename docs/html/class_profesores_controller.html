ProfesoresController::__construct ( )

Constructor del controlador.

Incluye el archivo init.php para iniciar la sesión y verificar la autenticación del usuario. Si el usuario no está autenticado, se redirige a la página de login. Luego, incluye el modelo de profesores y lo instancia.

ProfesoresController::store ( )

Procesa la solicitud para almacenar un profesor.

Dependiendo del valor del campo 'accion' enviado mediante POST, llama al método correspondiente para insertar o actualizar un profesor. Si el método HTTP no es POST, se muestra un mensaje de error.
