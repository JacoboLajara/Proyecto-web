◆ deleteModulo()

CursosModel::deleteModulo ( $id_modulo)

Elimina un módulo y sus relaciones asociadas de la base de datos.

Primero elimina las unidades formativas asociadas al módulo, luego elimina la relación en la tabla curso_modulo, y finalmente elimina el módulo de la tabla modulo.

Parameters
string$id_moduloIdentificador del módulo a eliminar.
Returns
bool True si la eliminación fue exitosa, false en caso contrario.

◆ getCursosConModulosYUnidades()

CursosModel::getCursosConModulosYUnidades ( )

Obtiene todos los cursos junto con sus módulos y unidades formativas.

Realiza una consulta a la tabla curso para obtener los cursos, luego obtiene los módulos asociados a cada curso y, finalmente, las unidades formativas correspondientes a cada módulo.

Returns
array Array de cursos, cada uno con sus módulos y unidades formativas.

◆ insertCursoWithModulesAndUnits()

CursosModel::insertCursoWithModulesAndUnits ( $curso,
$modulos,
$unidades )

Inserta un curso junto con sus módulos y unidades formativas.

Ejecuta una transacción que inserta un curso, luego los módulos asociados y finalmente las unidades formativas de cada módulo.

Parameters
array$cursoDatos del curso. Debe incluir 'nombre', 'codigo', 'duracion', 'tipo_curso' y 'precio'.
array$modulosArray de módulos. Cada elemento debe incluir 'nombre', 'codigo' y 'duracion'.
array$unidadesArray asociativo de unidades formativas. La clave debe ser el código del módulo y el valor, un array de unidades.
Returns
bool True si la transacción fue exitosa, false en caso de error.
