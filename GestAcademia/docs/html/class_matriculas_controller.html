◆ __construct()

MatriculasController::__construct ( )

Constructor del controlador.

Incluye el archivo init.php para iniciar la sesión y verificar si el usuario está autenticado. Si el usuario no está autenticado, se redirige a la página de login. Luego, se instancia el modelo de matrículas.

MatriculasController::getAlumnosMatriculados ( )

MatriculasController::procesarBajaAlumnos ( )

Procesa la baja de alumnos matriculados en un curso.

Lee los datos en formato JSON que deben incluir el ID del curso y una lista de alumnos. Llama al modelo para dar de baja a los alumnos indicados y devuelve una respuesta en JSON indicando el éxito o el fallo de la operación.

Returns
void

◆ store()

MatriculasController::store ( )

Procesa la matriculación de alumnos en un curso.

Lee los datos enviados en formato JSON desde la solicitud HTTP, valida que se reciban el ID del curso y la lista de alumnos, y para cada alumno verifica si ya está matriculado. Si no lo está, inserta la matrícula. Se responde en formato JSON indicando éxito o error.

Returns
void
