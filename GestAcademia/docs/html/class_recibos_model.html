Obtiene los recibos filtrados según los criterios.

Permite filtrar recibos utilizando parámetros opcionales: búsqueda parcial por apellido, año y mes de emisión, ID del curso y, opcionalmente, solo aquellos recibos en estado 'Pendiente'.

Parameters
string$apellido1Búsqueda parcial por el primer apellido del alumno.
string$anioAño para filtrar la Fecha_Emision (se evalúa con YEAR()).
string$mesMes para filtrar la Fecha_Emision (se evalúa con MONTH()).
string$cursoFiltrado por el ID del curso.
bool$pendientesSi es true, se filtran solo los recibos en estado 'Pendiente'.
Returns
array Arreglo asociativo con los recibos que cumplen los criterios.

◆ getTodosRecibos()

RecibosModel::getTodosRecibos ( )

Obtiene todos los recibos.

Esta función prepara una consulta para obtener todos los recibos. Se utiliza un parámetro llamado $pendientes, el cual se vincula mediante bind_param, aunque la consulta no contiene un marcador de posición para este parámetro.

Parameters
string$pendientesParámetro utilizado para filtrar (no se emplea en la consulta SQL).
Returns
array Array asociativo con todos los recibos.
