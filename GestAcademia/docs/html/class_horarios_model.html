◆ asignarHorario()

HorariosModel::asignarHorario ( $aula,
$curso,
$dia,
$hora_inicio,
$hora_fin )

Asigna un horario a un aula y curso específicos.

Primero verifica si existe un solapamiento en la franja horaria indicada para el aula y día especificados. Si se detecta solapamiento, devuelve un mensaje de error con la franja horaria ocupada y el curso asignado. Si no hay solapamiento, inserta el nuevo horario en la base de datos.

Parameters
int$aulaIdentificador del aula.
string$cursoIdentificador del curso.
string$diaDía en el que se asigna el horario.
string$hora_inicioHora de inicio en formato HH:MM:SS.
string$hora_finHora de fin en formato HH:MM:SS.
Returns
array Array con claves "success" (bool) y "message" (string) indicando el resultado de la operación.
