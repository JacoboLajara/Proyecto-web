◆ getNotificacionesPorAlumno()

NotificacionModel::getNotificacionesPorAlumno ( $idAlumno)

Obtiene las notificaciones enviadas a un alumno.

Ejecuta una consulta en la tabla Notificacion_Alumno para obtener los mensajes y la fecha de envío de las notificaciones dirigidas a un alumno, ordenados de forma descendente.

Parameters
string$idAlumnoIdentificador del alumno.
Returns
array Array asociativo con las notificaciones o un array vacío en caso de error.
