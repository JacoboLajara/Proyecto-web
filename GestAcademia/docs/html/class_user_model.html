◆ getAllUsersWithRoles()

UserModel::getAllUsersWithRoles ( )

Obtiene todos los usuarios junto con sus roles.

Realiza una consulta que une la tabla "Usuario" con las tablas "Alumno", "Profesor" y "Personal_No_Docente" (usando LEFT JOIN para cada una) para obtener el nombre, apellidos y rol de cada usuario. Se utiliza COALESCE para obtener el nombre y apellidos de la tabla que contenga datos.

Returns
array Un arreglo asociativo con los datos de los usuarios y sus roles.
