◆ __construct()

AlumnosController::__construct ( )

Constructor del controlador.

Incluye el archivo init.php para iniciar la sesión y verificar la autenticación del usuario. Si el usuario no está autenticado, se redirige al login. Finalmente, se instancia el modelo de alumnos.

◆ generarListadoAlumno()

AlumnosController::generarListadoAlumno ( $idAlumno)

Genera un listado detallado del alumno, incluyendo los cursos en los que está matriculado.

Utiliza la librería TCPDF para generar un PDF con la información detallada del alumno. Si no se encuentra información para el alumno, se termina la ejecución con un mensaje de error.

Parameters
string$idAlumnoIdentificador del alumno.
